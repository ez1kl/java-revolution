Java Revolution

L'Ombre de la Guillotine

À propos du projet

Java Revolution est un RPG narratif où vous incarnez un personnage dans la France révolutionnaire de 1789. Le jeu propose un système de combat au tour par tour, des choix qui influencent l'histoire, et une narration dynamique chargée depuis des fichiers JSON.
